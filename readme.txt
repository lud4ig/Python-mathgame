How to Run our Game

Prerequisite: You must have python installed on your machine.

For Windows users:
1. Navigate to the gamecreators folder (you should be inside this folder if you are reading this readme.txt file)
2. Right click anywhere in the folder (but not on the files itself) and click on "Open in Terminal"
3. Type "python main.py" and hit enter
4. The game will start running. Enjoy the game and music!

For Mac users:
All steps are the same, except for step 2, open the mac terminal instead.
